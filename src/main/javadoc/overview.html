A plugin for Netbeans to define a project type to create file(s) using a series of text merges, transformations and substitutions.

It runs a user definable pipeline of actions, controlled by parameters, communicating between actions with files in the standard filesystem, or a memory based file system.

Currently actions include:

  • Markdown processing (markdown to html fragments)
  • Substitute/Merge (insertion of file content or parameter strings into a template file)
  • Freemarker template language processing
  • XSLT processing (xml to xml as directed by an xslt stylesheet)
  • FOP processing (fo-xsl to pdf)
  • Creation of an html img tag suitable for use in a reactive modelled html page (using a srcset attribute)
  • Ability to run any OS utility program

Typical use cases are:

  • Creation of pdf documents (books/reports etc) from markdown sources
  • Creation of web sites, consisting of a set of static web pages, built from various sources (markdown , html templates, html fragments, parameters)
